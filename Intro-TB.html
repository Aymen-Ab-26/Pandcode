Introduction Tableaux

Le tableau

Le tableau est un type de variable. Il est défini comme une collection de valeurs du même type.
On peut l’imaginer comme une liste de cases alignées les unes à côté des autres, comme ceci : Il est très utile pour stocker plusieurs valeurs, par exemple : les températures pendant un mois.
Chaque case du tableau possède un numéro et une valeur.
On appelle ce numéro l’indice : il correspond au rang de la case.
En programmation, es indice commencent presque toujours à 0. l Donc :
la première case du tableau a l’indice 0
la suivante a l’indice 1, puis 2, 3, etc., jusqu’à N−1 (où N est le nombre total de cases dans le tableau)
